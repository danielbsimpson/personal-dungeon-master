# D&D 5e SRD — Core Rules

*Source: System Reference Document 5.1 (CC-BY-4.0, Wizards of the Coast)*

---

## Ability Scores

Every creature has six ability scores that represent its basic attributes.

| Ability | Governs |
|---|---|
| **Strength (STR)** | Physical power, melee attacks, carrying capacity |
| **Dexterity (DEX)** | Agility, reflexes, ranged attacks, AC (light/no armor) |
| **Constitution (CON)** | Endurance, hit points |
| **Intelligence (INT)** | Reasoning, memory, Wizard spellcasting |
| **Wisdom (WIS)** | Perception, insight, Cleric/Druid/Ranger spellcasting |
| **Charisma (CHA)** | Force of personality, Bard/Paladin/Sorcerer/Warlock spellcasting |

### Ability Score Modifiers

| Score | Modifier |
|---|---|
| 1 | −5 |
| 2–3 | −4 |
| 4–5 | −3 |
| 6–7 | −2 |
| 8–9 | −1 |
| 10–11 | +0 |
| 12–13 | +1 |
| 14–15 | +2 |
| 16–17 | +3 |
| 18–19 | +4 |
| 20–21 | +5 |
| 22–23 | +6 |
| 24–25 | +7 |
| 26–27 | +8 |
| 28–29 | +9 |
| 30 | +10 |

**Formula:** Modifier = ⌊(Score − 10) / 2⌋

## Proficiency Bonus

A proficiency bonus applies to attack rolls, ability checks, saving throws, and spell save DCs where the character is proficient.

| Character Level | Proficiency Bonus |
|---|---|
| 1–4 | +2 |
| 5–8 | +3 |
| 9–12 | +4 |
| 13–16 | +5 |
| 17–20 | +6 |

- A character adds their proficiency bonus once to a roll they are proficient in.
- A character never adds their proficiency bonus twice to the same roll (no stacking).
- Expertise doubles the proficiency bonus for a specific skill (Rogue and Bard feature).

## Ability Checks

When a character attempts a task with a chance of failure:

1. The DM calls for an ability check and sets a **Difficulty Class (DC)**.
2. The player rolls **1d20 + ability modifier** (+ proficiency bonus if proficient).
3. A result **equal to or higher** than the DC succeeds.

### Typical DCs

| Task Difficulty | DC |
|---|---|
| Very Easy | 5 |
| Easy | 10 |
| Medium | 15 |
| Hard | 20 |
| Very Hard | 25 |
| Nearly Impossible | 30 |

### Contests

When two creatures oppose each other directly, both make ability checks. The higher result wins. On a tie, the status quo prevails (the attacker fails, the resisting creature succeeds, etc.).

## Saving Throws

A saving throw represents a sudden physical or mental effort to resist an effect.

- Roll **1d20 + ability modifier** (+ proficiency bonus if proficient in that saving throw).
- Each class is proficient in two saving throws (defined in class features).
- A result equal to or higher than the DC set by the effect succeeds.

---

## Passive Scores

A **passive check** represents routine, ongoing awareness or activity without active effort.

**Passive score = 10 + all modifiers that normally apply to the check**

- **Passive Perception** is the most common. The DM uses it to determine whether a character notices something without actively searching.
- Advantage on a check adds +5 to the passive score; disadvantage subtracts 5.

---

## Advantage and Disadvantage

**Advantage:** Roll 2d20 and use the **higher** result.

**Disadvantage:** Roll 2d20 and use the **lower** result.

**Rules:**
- Multiple sources of advantage do not stack — two sources of advantage is still one extra die.
- If a creature has both advantage and disadvantage simultaneously (regardless of how many sources of each), the two cancel out and the creature rolls a single die normally.
- Any feature that says a creature "can't have advantage" overrides all sources of advantage.

## Hidden Rules (DM Reference)

- The DM decides when a check is needed. If a task has no chance of failure, or if the outcome is certain regardless of the roll, no check is required.
- The DM should not ask for a check unless both success and failure lead to interesting outcomes.
- A natural 1 on an ability check is not an automatic failure (unlike attack rolls and saving throws in some rules; check only matters vs. DC).
- A natural 20 on an ability check is not an automatic success — it simply gives the highest possible result on that die.
- For attack rolls and death saving throws: a natural 1 is always a failure; a natural 20 is always a success.
